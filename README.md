# ShareSphere – Smart Community Resource Sharing Platform

## Project Overview
ShareSphere is a MERN Stack based web application developed to manage and organize community resources efficiently. The application allows users to add, view, update, and delete resources dynamically.

This project demonstrates CRUD operations using React, Node.js, Express.js, MongoDB, and Mongoose.

---

## Features
- Add new resources
- View all resources
- Update existing resources
- Delete resources
- Form validation
- MongoDB database integration
- Responsive and professional UI

---

### Backend
